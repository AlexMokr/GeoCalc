О портале

Геопортал "Благополучие человека" разработан в рамках проекта Уральского федерального университета и направлен на поддержку реализации ключевых инициатив исследователей, бизнеса, органов власти в части мониторинга и анализа индикаторов, определяющих качество жизни в различных районах города. Портал наполняется датасетами с геопривязанными данными по разным категориям благополучия. В тестовой версии на основе собранных данных представлен функционал для микрорайона Уралмаш в Екатеринбурге.

В основе портала – единая база данных, которая позволит получать оперативную информацию по различным сферам жизни в городе. На портале для скачивания доступны датасеты с агрегированными показателями по домам, квартирам, инфраструктуре Уралмаша, муниципальные данные по Екатеринбургу. Данные по жителям предоставляются через форму заказа. По мере проведения исследований научными группами университета формируются новые датасеты с показателями, отражающими качество жизни: рынок труда, образование, медицина, эмоциональное состояние и др.

В перспективе исследователи и разработчики, используя формируемые наборы данных, могут проводить сравнительные исследования и создавать технологические продукты, разрабатывать сервисы и модели, полезные для жителей, бизнеса и органов власти.  Например, застройщики могут использовать данные, дашборды и карты для проведения предпроектного анализа территории, органы власти – для мониторинга и оперативного решения городских задач, а крупный и малый бизнес для разработки рекламных кампаний, маркетинговых и коммуникационных стратегий.
